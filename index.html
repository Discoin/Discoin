Ever wanted your hard-earned currency to go into something useful?

Discoin brings that problem to an end!

About

Discoin is a centralized currency for Discord bots, by centralizing the currencies we allow you to transfer your hard-earned coins fom one bot to another! (That is, if that bot supports it)

How does it work?

When you make a request for a credit transfer, the bot makes a request to our API and proceeds to transfer the credits using our API

How do I add this to my bot?!

You're a bot dev, huh? It's really simple! Here are the steps!

  1. Ask austinhuang#1076 for a token to the service
  2. Once you have a token, make a command to transfer credits/your currency
  3. To use our API make the command do a GET request to ENDPOINT_URL_CHANGE_THIS/transaction/USERID/AMMOUNT/CURRENCY
  4. Your tax rate is calculated by our API so you do not need to do anything else on your end
  5. Welcome to our centralized currency network!
